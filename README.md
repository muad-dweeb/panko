### Working Features

- Command prefix triggers: `!panko ` | `$`
- Add 5 platinum: `!panko +5pp`
- Subtract 10 gold: `$-10gp`
- View current funds: `$show`
- View this party's raw JSON data: `$raw`
- List available commands: `$help`
- More about Panko!: `$about`
